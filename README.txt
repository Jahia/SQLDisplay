This module - from Oshyn Team - displays the results performed by queries and it contains the capability to paging the results.

It offers the following custom fields:
1) selectStatement: Field to enter the sql query
2) dataSource: the data source that is going to be used to execute  the query
3) pageSize: the number of items per page and
4) maxRows: Max numbers of rows that is going to be showed in the results
